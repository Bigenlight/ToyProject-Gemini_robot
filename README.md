# ToyProject-Gemini_robot

![alt text](image-2.png)

A simple project for controlling a manipulator using the Gemini-3-flash-review model.

https://youtu.be/YGJHPRCHNgE

## Prerequisites

`git clone git@github.com:Bigenlight/ToyProject-Gemini_robot.git`

`cd ToyProject-Gemini_robot/`

`conda env create -f environment.yaml`

`conda activate gemini_robot`

## Run
- Needs API key on `../api_key.txt`.

<br>

`python main_agent_sequence.py`
